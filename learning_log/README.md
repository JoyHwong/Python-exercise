## 部署项目

建立虚拟环境

```python
python -m venv ll_env
```

激活虚拟环境

```python
source ll_env/bin/activate
```

安装Django

```python
pip install Django
```

创建数据库

```python
python manage.py migrate
```

查看项目

```python
python manage.py runserver
```
